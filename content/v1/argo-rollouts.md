# Definition
- [[k8s|쿠버네티스]]의 배포 전략을 관리하는 도구


# Origin
- 아르고 롤아웃의 어원은 [[argo-proj#Origin|아르고 프로젝트]]에서 찾을 수 있다.


# Objectives
- 배포 전략을 통해 애플리케이션의 가용성을 높이고, 롤백 및 모니터링 기능을 제공한다.


# Example
- Canary 배포 및 Blue-Green 배포 전략을 사용하여 새로운 버전을 안전하게 배포한다.
